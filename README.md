# puly
基于[ Image3D.js / core ](https://hai2007.gitee.io/image3d/index.html#/core)开发，底层依赖webgl实现，通过配置和简单的方法调用，可以快速实现三维数据可视化和VR效果等。

## 问题或交流
使用的时候遇到任何问题或有好的建议，请点击进入[issue](https://github.com/hai2007/puly/issues)！
